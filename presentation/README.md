# CalmCue Presentation Deck

Hackathon presentation for **CalmCue** (Sensory Overload Shield).

## Quick start

1. Open `index.html` in a modern browser (Chrome or Firefox recommended).
2. Press **F** for fullscreen.
3. Use **arrow keys** or **space** to advance; **Esc** for slide overview.

## Files

- **index.html** — Main slide deck (Reveal.js, CalmCue theme). No build step; open directly.
- **demo-script.md** — 3-minute demo script and speaker notes per slide.
- **README.md** — This file.

## Presenting

- Rehearse with `demo-script.md` to hit ~3 minutes (including a short live demo).
- For a “presenter view” with notes: open `demo-script.md` on a second screen or print the timing table.
- To export PDF: in the browser, print (Ctrl+P / Cmd+P) and choose “Save as PDF”; use landscape and “Background graphics” if available.

## Design

- CalmCue colors: teal/sage and cream background.
- One main idea per slide; sponsor tech and impact are clearly called out for judges.

Good luck at the hackathon.
